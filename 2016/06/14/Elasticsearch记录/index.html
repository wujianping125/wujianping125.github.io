发表于 2016-06-14   |   分类于 elasticsearch   |  

简介

Elasticsearch是一个基于Lucene构建的开源分布式搜索服务器。
官网：https://www.elastic.co/
优点：
完全支持 Apache Lucene 的接近实时的搜索。
Elasticsearch是分布式的，自动分配的写入机制。
易扩展，容错能力强。

缺点：
不支持事务
无权限管理（shield）

常用插件

Shield

Elasticsearch 没有自带的权限控制，需要安装Shield(需要购买，试用一个月过期不影响功能但head首页不能查看)

Shield2.0以上安装

Shield1.X 安装

启动Elasticsearch后，添加admin用户

head

一款H5的数据查看客户端。
安装

查看地址：http://localhost:9200/_plugin/head/
默认浏览器查看、curl命令端口为9200，9300为后台java程序端口。

bigdesk

状态查看客户端，可以查看当前Elasticsearch性能、cup占用、jvm内存等。
安装

查看地址：http://localhost:9200/_plugin/bigdesk/

ik-analyzer

常用的中文分词器

  1. git clone https://github.com/medcl/elasticsearch-analysis-ik.git
  2. mvn clean install（前提是配置了maven）。
  3. 将target中的elasticsearch-analysis-ik-xxx.jar复制到${esroot}/plugins/ik/。
  4. 复制elasticsearch-analysis-ik(git repository)中config目录下ik内容到{esroot}/config
